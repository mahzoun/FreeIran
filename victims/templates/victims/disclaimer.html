Disclaimer and ethics

This memorial is a documentation effort that relies on available sources and community submissions. We prioritize accuracy, transparency, and respect for families.

  • Unverified profiles are clearly labeled and may change as new evidence is reviewed.
  • Families can request corrections or removal at any time.
  • Private family contact information is never shown publicly.
  • Sources are listed for every profile whenever possible.

If you need to report an issue, use the correction form and include sources.
